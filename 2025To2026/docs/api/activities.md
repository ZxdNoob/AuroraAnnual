# 活动 API 文档

## 概述

活动模块提供活动管理、活动参与等功能。

**路径前缀**：`/api/activities`

## 接口列表

### 1. 创建活动（管理员）

**接口**：`POST /api/activities`

**描述**：创建新活动（仅管理员）

**认证**：需要（管理员权限）

### 2. 获取所有活动

**接口**：`GET /api/activities`

**描述**：获取所有活动列表（公开接口）

### 3. 获取当前进行中的活动

**接口**：`GET /api/activities/active`

**描述**：获取当前进行中的活动（公开接口）

### 4. 获取活动详情

**接口**：`GET /api/activities/:activityId`

**描述**：获取指定活动的详细信息（公开接口）

### 5. 参与活动

**接口**：`POST /api/activities/participate`

**描述**：参与活动

**认证**：需要

### 6. 获取我的参与记录

**接口**：`GET /api/activities/my/participations`

**描述**：获取当前用户的活动参与记录

**认证**：需要
