# 勋章 API 文档

## 概述

勋章模块提供勋章列表、用户勋章、勋章统计等功能。

**路径前缀**：`/api/badges`

## 接口列表

### 1. 获取所有勋章列表

**接口**：`GET /api/badges`

**描述**：获取所有可获得的勋章列表

**认证**：需要

**查询参数**：

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| type | string | 否 | 勋章类型筛选 |
| rarity | string | 否 | 稀有度筛选 |

### 2. 获取我的勋章列表

**接口**：`GET /api/badges/my`

**描述**：获取当前用户已获得的勋章列表

**认证**：需要

### 3. 获取我的勋章统计

**接口**：`GET /api/badges/my/statistics`

**描述**：获取当前用户的勋章统计信息

**认证**：需要

### 4. 检查并授予符合条件的勋章

**接口**：`GET /api/badges/check`

**描述**：手动触发勋章检查

**认证**：需要
