# 道具 API 文档

## 概述

道具模块提供道具列表、道具使用、道具效果查询等功能。

**路径前缀**：`/api/items`

## 接口列表

### 1. 获取所有道具

**接口**：`GET /api/items`

**描述**：获取所有道具列表（公开接口）

### 2. 获取我的道具列表

**接口**：`GET /api/items/my`

**描述**：获取当前用户拥有的道具列表

**认证**：需要

### 3. 使用道具

**接口**：`POST /api/items/use`

**描述**：使用道具

**认证**：需要

### 4. 获取我的生效道具效果

**接口**：`GET /api/items/my/effects`

**描述**：获取当前用户生效的道具效果

**认证**：需要
