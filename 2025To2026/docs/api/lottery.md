# 抽奖 API 文档

## 概述

抽奖模块提供抽奖、抽奖历史、抽奖统计等功能。

**路径前缀**：`/api/lottery`

## 接口列表

### 1. 执行抽奖

**接口**：`POST /api/lottery/draw`

**描述**：执行抽奖操作，每次消耗 50 积分

**认证**：需要

**响应示例**（200 OK）：

```json
{
  "prize": {
    "id": "1",
    "name": "小红包",
    "type": "RED_PACKET",
    "value": 10
  },
  "pointsSpent": 50,
  "pointsEarned": 10
}
```

### 2. 获取我的抽奖历史记录

**接口**：`GET /api/lottery/history`

**描述**：获取当前用户的抽奖历史记录

**认证**：需要

### 3. 获取我的抽奖统计

**接口**：`GET /api/lottery/statistics`

**描述**：获取当前用户的抽奖统计信息

**认证**：需要

### 4. 获取所有奖品配置

**接口**：`GET /api/lottery/prizes`

**描述**：获取所有奖品配置（公开接口）

**认证**：不需要
